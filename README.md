Prompt -

Create a modern, clean, and visually attractive personal portfolio website using HTML, CSS, and JavaScript — ALL IN A SINGLE FILE (index.html). Do NOT create separate CSS or JS files. Embed CSS inside <style> and JavaScript inside <script>.
Requirements:

1. General Design:
- Use a professional and aesthetic UI/UX design.
- Include both Light Mode and Dark Mode with a toggle button (default to system preference).
- Use smooth animations (fade, slide, hover effects).
- Fully responsive design (mobile, tablet, desktop).
- Use modern fonts (Google Fonts like Poppins or Inter via CDN).
- Use a consistent and attractive color palette.

2. Structure (Single File Only):
- Everything must be inside index.html
- CSS must be inside <style> tag
- JavaScript must be inside <script> tag
- Keep code clean and well organized with comments

3. Header / Navbar:
- Sticky navigation bar
- Menu items:
  Home | About | Experience | Education | Projects | Skills | Contact
- Add a "Download CV" button (link: /cv/my-cv.pdf)
- Add Dark/Light mode toggle button
- Mobile hamburger menu

4. Hero Section:
- Placeholder name and title (easy to replace later)
- Short intro/tagline
- Profile image (load from img/profile.jpg)
- Buttons: Contact Me, View Projects
- Add typing animation effect

5. About Me Section:
- Short professional summary
- Personal details (location, email, etc.)
- Add comments showing where to edit content

6. Experience Section:
- Timeline or card layout
- Job title, company, duration, description

7. Education Section:
- Degree, institution, year
- Clean modern layout

8. Skills Section:
- Progress bars or skill cards
- Categories (Frontend, Backend, Tools)

9. Projects Section:
- Responsive grid layout
- Each project includes:
  - Image (from img folder)
  - Title
  - Description
  - Tech stack
  - GitHub / Live demo links

10. Contact Section:
- Contact form (Name, Email, Message)
- Social media links (GitHub, LinkedIn, etc.)

11. Footer:
- Simple and clean footer
- Social icons
- Copyright text

12. Features:
- Smooth scrolling
- Scroll animations (CSS or JS)
- Hover effects
- Scroll-to-top button
- Section titles with stylish design
- Use CSS variables for theme (light/dark)

13. CV Integration:
- Add "Download CV" button in navbar
- Use placeholder: /cv/my-cv.pdf

14. Image Handling:
- Use images from /img folder only
- Example:
  <img src="img/profile.jpg" alt="Profile">
15. Data Integration:
- Use placeholder content
- Add clear comments like:
  <!-- Replace with your name -->
  <!-- Replace with your experience -->
- Make it easy to update after adding my CV
16. Code Quality:
- Clean, readable, well-commented code
- Use semantic HTML
- Avoid unnecessary complexity
Goal:
The final output should be a single index.html file that looks modern, elegant, professional, and suitable for a software developer portfolio, with all features working perfectly.
